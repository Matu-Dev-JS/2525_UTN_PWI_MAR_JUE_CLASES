Listar todas las cosas que DEBEMOS SABER de JS, HTML, CSS para poder seguir cursando sin problemas

HTML
- Formularios: form, input, label, button submit
- HTML Semantico: nav, header, footer, section, article, main, aside
- Rutas relativas: ../, ./
- Heading

CSS
- Propiedades comunes:
    - Box Modeling
        - padding
        - margin
        - border
        - box-sizing
    - Texto

- Flexbox: flex-direction, justify-content, aling-items, gap

- Responsive: @media, flexbox aplicado a responsive, medidas relativas (px, vw, vh, %)
    - La mayoria de scrolls horizontales, desbordamientos ETC se dan cuando cierto elemento hijo es mas grande que el padre
    - Usen las mediaqueries que necesiten, no se limiten en usar 10 o 20 mediaqueries de ser necesario
    - Inspector y el modo responsive del navegador son temas fundamentales para debbuguear un css

- Position: 
    - Relative + Absolute
    - Relative
    - Fixed
    - Sticky

JS:
- Tipos de dato
- Operadores logicos (&& y || y !)
- Comparadores (>, <, >=, <=, ==, ===, !=, !==)
- Operadores aritmeticos (+, -, *, /)
- Variables (Con saber let basta) y (alcance y ambito)
- Bloques de codigo (basico)
- IF, ELSE
- FOR
- FUNCTION (parametros y retorno tambien)
- Arrays (con los metodos (push y splice almenos))
- Objetos (como crear y actualizar un objeto)
    - let persona = {nombre: 'pepe'}
    - console.log(persona.nombre)
- Recomiendo saber FOR OF
